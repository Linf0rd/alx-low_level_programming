This project involves solving a series of tasks using C programming language. The tasks are aimed at improving the understanding of arithmetic, logical, relational and boolean operators in C, as well as the use of char, int and unsigned int variable types, if and if...else statements, and while loops.

The following are the tasks that were completed:

0-positive_or_negative.c - A C program that prints whether a randomly generated number is positive or negative.
1-last_digit.c - A C program that prints the last digit of a randomly generated number.
2-print_alphabet.c - A C program that prints the alphabet in lowercase using only the putchar function twice.
3-print_alphabets.c - A C program that prints the alphabet in lowercase, followed by the alphabet in uppercase.
4-print_alphabt.c - A C program that prints the alphabet in lowercase except for the letters q and e.
5-print_numbers.c - A C program that prints all single digit numbers of base 10 starting from 0.
6-print_numberz.c - A C program that prints all single digit numbers of base 10 starting from 0 using only the putchar function twice.
7-print_tebahpla.c - A C program that prints the lowercase alphabet in reverse.
8-print_base16.c - A C program that prints all the numbers of base 16 in lowercase.
9-print_comb.c - A C program that prints all possible combinations of single-digit numbers.
10-print_comb2.c - A C program that prints the numbers from 00 to 99.
100-print_comb3.c - A C program that prints all possible different combinations of two digits.
101-print_comb4.c - A C program that prints all possible different combinations of three digits.
The number of times to use putchar function in each program is restricted to a maximum limit.
